# 프로젝트 Description
## 프로젝트 목표
- 패스오더의 검색 기능 중 검색어 입력-페이지 이동 과정을 자동화하여 반복 테스트를 용이하게 할 것.
- 단순한 click 액션 뿐만 아니라 실제 검색 결과에서 매장 상세 정보 화면으로 이동하여 검색어가 포함되어 있는지 확인하는 과정도 추가하여 logging 할 것.

## 테스트 상세 과정
### 검색 수행
- 메인 화면 → 검색 화면으로 이동
- 검색창에 keyword 입력
- keyword 검색 결과의 첫번째 매장 상세 페이지로 이동

### 매장 상세 정보 화면에서 검색어 확인
- 매장 상세 정보 화면의 매장 이름에 keyword가 포함되어 있는지 확인
- 매장 상세 정보 화면 → 검색 화면으로 이동
- 검색 화면 → 메인 화면으로 이동

## 테스트 환경
- 자동화 도구: Appium 2.5.0
- 작성 언어: Python
- 테스트 대상: Android
